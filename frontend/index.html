What should my file look like?

Almost any CSV or Excel works — column names don't need to be exact, and extra columns are kept as context. For the best results:

  • First row = column headers, then one lead per row.
  • Include a first name column (e.g. First Name) and a company column (e.g. Company or Organization). Some campaigns also use an email column.
  • For company-only lists with no contact name, the greeting may use the company name — add a first-name column to personalise it.
  • Use a single sheet, no merged cells or blank header rows.
  • Don't include Subject_Touch / Body_Touch columns — those are generated for you.
  • Formats: .csv or .xlsx, up to 50 MB.
